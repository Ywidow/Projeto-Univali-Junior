Instruções

  • Este site se caracteriza como um estilo de "mercado", onde foi feito atráves das exigências fornecidas pela empresa

  • Na latera esquerda, é possível ver uma "aba de navegação", onde será seu principal "mapa", mapa se localizar e acessar as páginas principais

  • No botão "Home", você será encaminhado para esta exata página

  • No botão "List", você será encaminhado para a página de listagem dos itens do mercado. Nesta página de listagem, você poderá ver todos os itens dentro do seu mercado, e poderá também alterar qualquer item, ou atualizar as informações do item

  • No botão "Form", você será encaminhado para a seção desta página onde há o formulário em questão. Neste formulário, você poderá adicionar um item ao seu mercado
